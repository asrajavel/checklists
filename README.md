# Hometown Trip Checklists

Printable A4 checklists for hometown trip preparation.

- `last-week.html` — Week-before tasks (cleaning, washing, etc.)
- `last-day.html` — Last day tasks (6 PM onwards)
- `styles.css` — Shared styles

**View/Print:** Double-click any `.html` file, then Ctrl+P. Enable "Background graphics" in print settings.

**Edit items:** Add/remove `<div class="item">...</div>` lines in the HTML files.
